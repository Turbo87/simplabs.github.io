Our Tools

We focus on a set of framework and languages that we realize most of our projects with.

Ruby on rails® is an open-source web framework that's optimized for programmer happiness and sustainable productivity.

Ember.jsis a JavaScript framework for building ambitious applications in the browser. It works great with Ruby on Rails backends.

Swiftis Apple's new programming languages for iOS and OS X. It's both faster and more powerful than Objective-C.

While these three constitute our standard toolbox we also use a set of other tools and services

Our Process

We are enthusiastic about agile development practices and flexible processes. While we don't think planning is evil per se we don't overplan and communicate with our customers a lot. We usually recommend having a kick-off meeting to get the project going and from that on refine things as needed.
