Our Tools

We build on a standard set of framework and languages to realize most of our projects.

Ruby on rails® is an open-source web framework that's optimized for programmer happiness and sustainable productivity.

Ember.jsis a JavaScript framework for building ambitious applications in the browser. It works great with Ruby on Rails backends.

Swiftis Apple's new programming language for iOS and OS X. It's both faster and more powerful than Objective-C.

Our Process

We are enthusiastic about agile practices and a flexible development process. While we don't think planning is evil per se we don't overplan and communicate with our customers a lot. We usually recommend having a kick-off meeting to get the project going and from that on refine things as needed.
