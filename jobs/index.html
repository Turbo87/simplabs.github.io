Intern or Working Student in Munich

We are looking for a student who is interested in the web as a platform, is passionate about modern technologies and well crafted code and able to learn and apply new things quickly. Besides working on our client and open source projects which are built with the latest web technologies and frameworks like ES2015 (ES6), Ember.js and Elixir, they will also work on internal projects on their own authority.

We offer competitive compensation, the possibility to attend and speak at international conferences (like Ember Conf 2016 in Portland, USA which we are sponsoring), an experienced international team that maintains and contributes to some of the most popular Ember.js libraries and a nice office in Munich. We will provide you with a MacBook Pro and a Cinema Display to make sure you have a nice working setup.

We don't require any formal qualification. You should have basic experience with

  • HTML
  • JavaScript

Having heard of the following technologies would be a plus (even better if you already used some of them):

  • Ruby on Rails or other server side frameworks like express or play
  • Elixir/Erlang
  • JSON API
  • d3 or other frontend graphics libraries
  • Websockets
If you have a github account please include that in the application.

If you feel you are a good fit, send us a message to info@simplabs.com or contact us on twitter at @simplabs. Please make sure to include your github account.
