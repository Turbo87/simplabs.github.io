Open Source

Highlight Source Issues

Highlight is a plugin for Ruby on Rails that provides syntax highlighting for source code in more than 30 languages.

Excellent Source Issues

Excellent is a source code analysis gem for Ruby that combines roodi, reek and flog and also adds some Ruby on Rails specific checks.

Reportable Source Issues

Reportable is a plugin for Ruby on Rails that enables you to generate reports and sparklines from your models with just one line of code.

Melbourne Source Issues

Melbourne is an extraction of Rubinius' parser component into a standalone gem. It can be user to generate Abstract Syntax Trees from arbitrary Ruby code.

Glider Source Issues

Glider is a simple Javascript content glider based on prototype and script.aculo.us. You can see it in action on simplabs.com (just click here or in the top menu).
